# String Casing

This library supports both runtime and design-time utilities for standard casing strategies such as:

- `PascalCase`
- `camelCase`
- `snake_case`
- `kebab-case`

## Runtime Utilities

- `toPascalCase()`
- `toCamelCase()`
- `toSnakeCase()`
- `toKebabCase()`

## Type Utilities

- `PascalCase<T>`
- `CamelCase<T>`
- `SnakeCase<T>`
- `KebabCase<T>`
